Sviluppo di un'applicazione web per scambio di commenti tra più utenti.
Tutti i commenti sono visibili nella homepage scritti da più utenti.
Nella parte comments sono visibili i commenti con i rispettivi nomi.
Per aggiungere, modificare o cancellare i commenti, gli utenti devono essere registrati o fare il login.
